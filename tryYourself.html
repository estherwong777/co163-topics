Try It Yourself!

Exploring BRDF with Walt Disney Animation Studios

Walt Disney has created a tool for exploring different types of BRDF. The software allows the developlemt and analysis of the BRDFs. It is really interactive and the graphs and visualization update real time as various parameters are changed.

Software can be downloaded from here

Mitsuba - A rendering tool

Mitsuba is an open source renderer that was created as an academic research project. You can use this to experiment with rendering different types of materials.

Mitsuba can downloaded from here
